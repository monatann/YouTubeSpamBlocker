# YouTubeSpamBlocker
VTuberのコメ欄に出てくる糞Bot自動検出&amp;コメント非表示をするUserScriptです。<br>
一定の文字数以上の名前で同じコメントすると誤検出するのでそこだけは注意してください。<br>
自動通報&ブロック機能は私の技量だと難しいです。<br>
このファイルを使用して何かあったとしても製作者のmonatannは何も責任は負いません。自己責任でお願いします。
